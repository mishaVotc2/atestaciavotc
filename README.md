# Movie Library — Личная кинотека

**Автор:** Вотчинников Михаил


## Описание

GUI-приложение для хранения коллекции фильмов. Позволяет добавлять, удалять, фильтровать по жанру и году, сохраняет данные в JSON-файл.

## Технологии

- Python 3
- Tkinter (встроенный GUI)
- JSON для хранения данных

## Установка и запуск

1. Убедитесь, что установлен Python 3.6+
2. Скачайте файл `movie_library.py`
3. Запустите:
   ```bash
   python movie_library.py
